# Card

Surface container. Compose with the card parts — all exported from the same package:
`CardHeader`, `CardTitle`, `CardDescription`, `CardContent`, `CardFooter`, `CardAction`.

```tsx
<Card>
  <CardHeader>
    <CardTitle>AI Readiness Audit</CardTitle>
    <CardDescription>A flat-fee diagnostic.</CardDescription>
  </CardHeader>
  <CardContent>…</CardContent>
  <CardFooter><Button>Get the Assessment</Button></CardFooter>
</Card>
```
